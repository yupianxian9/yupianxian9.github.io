Scoop使用指南

2024/10/17 技术

scoop是一款强大的Windows端包管理工具。

scoop官网: https://scoop.sh/

一、Scoop安装与使用

以下是 Scoop 的安装和使用教程，涵盖替换为国内 Gitee 镜像加速下载和将 Scoop 安装到特定文件夹：

1. 前置准备

  1. 安装 PowerShell 7 或更高版本（建议使用 PowerShell Core，安装速度更快）。

  2. 确保系统环境变量中包含 PowerShell。

  3. 安装 .NET Framework 4.5 或更高版本（Windows 10+ 通常已包含）。

  4. 检查系统是否开启脚本执行

    在 PowerShell 中运行：

    Get-ExecutionPolicy

    如果输出不是 RemoteSigned 或 Unrestricted，则运行以下命令：

    Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

2. 将 Scoop 安装到特定文件夹

默认情况下，Scoop 安装在用户目录的 C:\Users\<用户名>\scoop 文件夹下。如果需要指定其他路径，请按照以下步骤操作：

安装 Scoop 到指定目录

  1. 手动创建目标文件夹：
    例如，将 Scoop 安装到 D:\Scoop，先手动创建 D:\Scoop 文件夹。

  2. 设定安装路径

    在 PowerShell 中执行以下命令，将 SCOOP

    环境变量设置为自定义路径：

    $env:SCOOP='D:\Scoop'
[environment]::setEnvironmentVariable('SCOOP', $env:SCOOP, 'User')

  3. 安装：
    安装完成后，检查目标路径是否生成了 Scoop 的目录结构。

Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression

注意：若上述命令安装不了，可以使用下面的GitHub镜像进行安装

irm https://cdn.jsdelivr.net/gh/duzyn/scoop-cn/install.ps1 | iex

3. 替换为国内 Gitee 镜像加速下载

国内用户使用 Scoop 时可能因网络问题导致安装或下载软件缓慢。可以通过替换源为 Gitee 镜像来解决。

替换主 Scoop 源

  1. 切换到 Gitee 镜像源：

    #切换到南京大学镜像源
scoop config SCOOP_REPO https://mirror.nju.edu.cn/git/scoop-main.git
# 拉取新库地址
scoop update

  2. 修改默认 bucket（应用仓库）地址：

    已有的仓库需要先删除掉，然后才能够替换为国内源

scoop bucket rm main

将默认 bucket 指向国内镜像：

scoop bucket add main https://mirror.nju.edu.cn/git/scoop-main.git

替换额外的 bucket 源

Scoop 支持添加多个 bucket。例如，添加 extras 仓库的国内镜像：

scoop bucket add extras https://mirror.nju.edu.cn/git/scoop-extras.git

检查镜像生效情况：

运行以下命令查看 bucket 列表：

scoop bucket list

4. Scoop 的基本使用

以下是常用的 Scoop 命令和使用示例：

  1. 安装软件：
    • 安装单个软件：scoop install <app_name>，如scoop install git。
    • 安装指定版本软件：scoop install <app_name>@<version>。
    • 全局安装软件：scoop install <app_name> -g。
### 直接使用 scoop install 命令安装本地文件
将本地安装包放置在合适的位置
将需要安装的软件包（如 .msi、.exe、.zip 等格式的文件）放置在一个容易访问的目录中，例如 C:\LocalPackages。
使用 scoop install 命令安装
在 PowerShell 中运行以下命令，指定本地安装包的路径：

scoop install path C:\LocalPackages\your-package-file.exe

或者：

scoop install path C:\LocalPackages\your-package-file.msi

其中 C:\LocalPackages\your-package-file.exe 是本地安装包的完整路径。
  1. 卸载软件：
    • 卸载软件：scoop uninstall <app_name>。
    • 卸载软件并删除相关数据：scoop uninstall <app_name> -p。
  2. 搜索软件：
    • 使用自带搜索命令：scoop search <app_name>。
    • 安装scoop-search插件后使用：scoop install main/scoop-search，然后通过scoop-search <app_name>进行搜索。
  3. 查看已安装软件：scoop list。
  4. 更新软件：
    • 更新Scoop：scoop update。
    • 更新单个软件：scoop update <app_name>。
    • 更新所有软件：scoop update *或scoop update -a。
  5. 禁止更新软件：
    • 禁止更新：scoop hold <app_name>。
    • 解除禁止更新：scoop unhold <app_name>。
  6. 清理旧版本软件：scoop cleanup <app_name>。
  7. 清理安装文件：
    • 查看所有安装文件：scoop cache。
    • 删除指定软件安装文件：scoop cache rm <app_name>。
    • 删除所有安装文件：scoop cache rm *或scoop cache rm -a或scoop cache rm --all。

5.管理软件仓库（Bucket）

  1. 添加仓库：scoop bucket add <仓库名>，如scoop bucket add extras。
  2. 查看已添加仓库：scoop bucket list。
  3. 移除仓库：scoop bucket rm <仓库名>。

6.卸载scoop

scoop uninstall scoop
